## Libraries
- [LibEditMode](https://www.curseforge.com/wow/addons/libeditmode): Library that allows a developer to integrate their frames into the Edit Mode introduced in Dragonflight.
- [LibOpenRaid](https://www.curseforge.com/wow/addons/openraid/): Help to transmit data among players in the Raid and Group.
- [LibSharedMedia](https://www.curseforge.com/wow/addons/libsharedmedia-3-0): Handles all additional media for the AddOn.
